# Bellini College Pathways Explorer

A React + React Flow prototype that maps student entry points to programs and career outcomes.

## Quick Start

```bash
npm install
npm run dev
```

## Whatâ€™s Included

- Triage overlay (3-step questionnaire) that drives BFS highlighting.
- Left-to-right DAG with custom nodes, hover cards, and active edge styling.
- Success story modal tied to career outcomes.
- Mobile layout that stacks columns vertically.

## Data

Update `src/data.json` to add or adjust nodes and edges. Fields like `status`, `interests`, and `goals` are used by the filtering logic.
